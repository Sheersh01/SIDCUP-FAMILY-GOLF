# SIDCUP - FAMILY GOLF ⛳

A visually stunning and interactive front-end web clone of the Sidcup Family Golf website. This project focuses on building a modern, highly engaging user interface with custom cursor effects, smooth scrolling animations, and a polished design.

## 🌟 Features

- **Custom Interactive Cursor**: A unique, blurred cursor that follows the mouse and reacts to specific elements (like the navigation bar) for a premium feel.
- **Scroll Animations**: Dynamic animations powered by GSAP and ScrollTrigger that bring elements to life as you scroll down the page.
- **Responsive Navigation**: A sticky navigation bar that adapts its background and text colors based on scroll position.
- **Modern UI Design**: Beautiful typography, hover effects, and a dark/light thematic transition.
- **Video Hero Section**: An engaging full-screen background video on the landing page.

## 💻 Technologies Used

- **HTML5**: For structuring the web pages.
- **CSS3**: For styling, layout, and custom CSS animations.
- **JavaScript (Vanilla)**: For DOM manipulation and logic.
- **GSAP (GreenSock Animation Platform)**: For complex, high-performance animations.
- **GSAP ScrollTrigger**: For tying animations to the user's scroll position.
- **Remix Icons**: For scalable, lightweight icons.

## 🚀 How to Run

1. Clone or download this repository to your local machine.
2. Open the project folder.
3. Simply open `index.html` in your favorite web browser (e.g., Chrome, Firefox, Safari) to view the website. No server or build process is required!
